One thing I built.

01 / OVERVIEW

Customer Support RAG Chatbot

AI-Powered Customer Support Chatbot (RAG) — built with Python, FastAPI, LangChain, OpenAI/Gemini, and ChromaDB to answer customer questions from internal knowledge base documents.

Python FastAPI LangChain OpenAI / Gemini ChromaDB Docker
02 / THE PROBLEM

Customers couldn't find things.

Large knowledge bases meant delayed support responses, frustrated customers, and an overworked support team. Search was keyword-based and brittle. The bottleneck wasn't writing docs — it was finding them.

03 / THE APPROACH

Embed → Store → Retrieve → Generate.

Ingest — PDFs and text files chunked, embedded with OpenAI, stored in ChromaDB.

Retrieve — User query embedded, top-k semantic search, context window assembled.

Generate — Context + chat history fed to GPT/Gemini via LangChain. Answer with source citations.

04 / DEPLOY

Containerised and ready.

Implemented document ingestion, chunking, embedding generation, semantic retrieval, and REST APIs for conversational support flows. Containerized the solution with Docker for easier setup and deployment.
